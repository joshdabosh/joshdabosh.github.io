  • BANJO - Blazar And Neutrino Juxtaposition Operation

    A program that can accurately identify what blazar is most likely to have originated a collection of neutrinos, given a collection of those neutrino events. This was my partner and I's entry into the 2018-19 Montgomery County Science Fair. We won 2nd place in Physics, as well as a number of other community awards, such as awards from the American Nuclear Society and the Nuclear Regulatory Commitee.

    If you'd like to see our project in action, contact me! I'd be happy to show you what it can do. Currently, the GitHub repository is private.

  • E-Shuushuu scraper

    A program that scrapes images off the image board e-shuushuu.net. I originally planned for this to be a tool to help build my dataset of anime eyes, as a past project of mine was to build a Haar cascade that could recognize anime eyes. However, that project didn't work out as well as I'd hoped, and I was just left with a really efficient, multithreaded tool for downloading images quickly. It uses requests for downloading pages and BeautifulSoup 4 for webpage parsing.

  • AutoPack

    An automated backpack that I entered into the 2017-18 Montgomery County Science Fair as the leader of the project. The project won Honorable Mention in the Engineering Category. I presented this project at the 2017-18 Montgomery County Agricultural Fair, and won Grand Champion in the STEM Category. The code is run on a Raspberry Pi. Power and sensor signals are handled through a breadboard. The final "stage" of the project involved the usage of computer vision and ultrasonic sensors to provide autonomy when following targets.

  • PlusMinus

    A website based on a game I learned during a Game Theory classat a math camp. Players take turns marking either 1 or 2 dashes with a line, and the first player that cannot make another valid move loses. The site features a NoSQL database using Google's FireBase RealTime database, and is written in Flask. The site is hosted on Heroku, where all important credentials are kept as config vars as well as my local machine.

    An music class announcements website some friends and I put together.
