Projects

  • Sourcing Neutrinos to Blazars

    A program that can accurately identify what blazar is most likely to have originated a collection of neutrinos, given a collection of those neutrino events. This was my partner and I's entry into the 2018-19 Montgomery County Science Fair. We won 2nd place in Physics, as well as a number of other community awards, such as awards from the National Capital Astronomers and the Nuclear Regulatory Commitee..


    Please contact jwanggt@gmail.com if you are interested in the program.

    • PorchJokes

    A jokes board I originally for submissions to Mr. Porch's Joke of the Day. It is a Flask web application and I used an SSH request-forwarding service called serveo to get a subdomain. The MySQL server is run locally.

  • TPMSMusic

    A website some friends and I put together. Our band director had some issues with communication with parents and students, and was interested if some of her students could make a website for announcements and file sharing. The backend is written in Flask, and the site itself is hosted on Heroku. It uses a FireBase NoSQL database for both posts and user credentials.

  • AutoPack

    An automated backpack that I entered into the 2017-18 Montgomery County Science Fair. The project won Honorable Mention in the Engineering Category. I presented this project at the 2017-18 Montgomery County Agricultural Fair, and won Grand Champion in the STEM Category. The code is run on a Raspberry Pi. Power and sensor signals are handled through a breadboard.
