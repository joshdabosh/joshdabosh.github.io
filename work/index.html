Aweeb

A Discord bot that spams a whole bunch of anime pictures, downloaded with my E-Shuushuu downloader. I implemented a system so that only I could spam the anime pictures, preventing constant abuse. The bot does this by generating a string of 4 of the 100 most common words and then hashing them using the MD5 Algorithm. The bot then sends this key to my Private Messages and I can spam pictures using that key.
